Como usar o ProxyFlow Containers

Aprenda a configurar proxies por contexto no Firefox em poucos minutos.

1

Entendendo os contextos do Firefox

O Firefox separa as sessões em contextos. Cada contexto tem seus próprios cookies e pode usar um proxy diferente:

  • Padrão da extensão Base para tudo que não tem regra específica.
  • Sem container Abas normais abertas sem um container específico.
  • Janela privada Navegação privada. Ative a extensão em janelas privadas nas configurações do Firefox.
  • Containers Contextos separados que você cria (ex.: Trabalho, Pessoal, Compras).
2

Configurando o proxy padrão

Na seção "Geral e containers", configure o Padrão da extensão primeiro — ele serve de base para os outros contextos que usam "Herdar padrão":

ModoProxy manual
Endereçosocks5://127.0.0.1:1080

💡 Formatos aceitos: socks5://host:porta, http://host:porta ou só host:porta (padrão SOCKS5)

3

Modos disponíveis

Conexão direta

Acessa a internet sem nenhum proxy.

Proxy manual

Usa o proxy que você configurar.

Bloquear

Impede qualquer conexão neste contexto.

Herdar padrão

Segue o modo do "Padrão da extensão".

Usar Firefox

Usa as configurações de proxy do próprio Firefox.

4

Regras por site

Crie regras que têm prioridade sobre os containers. Úteis para forçar ou bloquear proxy em sites específicos:

Domínioyoutube.com
Subdomínios*.empresa.com
URL completahttps://api.site.com/*

⚡ Use "Todos os contextos" no campo "Aplicar em" para que a regra valha em qualquer container.

5

Testando seu proxy

Clique em "Testar" em qualquer perfil ou regra. A extensão vai:

  • Conectar via HTTPS usando o proxy configurado
  • Exibir o IP público visto pelo servidor (Cloudflare)
  • Mostrar a velocidade: rápido / médio / lento
  • Indicar o país da saída pelo código de localização
6

Proxies públicos rápidos

Não tem proxy? Use a busca automática de fontes públicas:

  • Escolha o protocolo — SOCKS5 é o mais anônimo
  • Clique em "⚡ Obter e testar"
  • A extensão busca, testa HTTPS real e salva os que funcionam
  • Use "Usar melhor salvo" para aplicar o mais rápido

⚠️ Proxies públicos são instáveis e podem ser monitorados. Nunca use para login, banco, e-mail ou dados sensíveis.

7

Preferências avançadas

  • Resolver DNS pelo proxy (SOCKS): o DNS é resolvido no servidor proxy, não no seu dispositivo. Aumenta privacidade e evita vazamentos de DNS.
  • Bloquear quando proxy inválido: se um contexto está configurado para proxy mas o proxy não é válido, a conexão é bloqueada em vez de cair na internet direta.
