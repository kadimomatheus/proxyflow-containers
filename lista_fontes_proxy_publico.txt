Este documento reúne as principais fontes públicas e gratuitas de proxies (HTTP, HTTPS, SOCKS4 e SOCKS5), ordenadas das mais recomendadas e rápidas até as secundárias.

--------------------------------------------------------------------------------
⚠️ AVISOS IMPORTANTES DE SEGURANÇA E USO
--------------------------------------------------------------------------------
1. PRIVACIDADE E RISCO: Proxies públicos/gratuitos são inerentemente instáveis, 
   lentos e podem ser controlados por agentes maliciosos para monitorar tráfego.
2. DADOS SENSÍVEIS: NUNCA trafegue dados bancários, e-mails pessoais, senhas, 
   WhatsApp Web ou sessões logadas por meio de proxies públicos.
3. PROTOCOLOS RECOMENDADOS: Dê preferência aos protocolos SOCKS5 e HTTPS para 
   navegação normal. HTTP simples não criptografa as requisições.
4. CRITÉRIO DE VELOCIDADE: Um bom proxy público deve carregar uma página HTTPS 
   estável (como https://example.com) em até 5 a 8 segundos.

================================================================================
💡 RECOMENDAÇÕES DE LÓGICA DE VALIDAÇÃO (TESTES DE CONEXÃO)
================================================================================
Para obter a melhor performance com a extensão:
1. Obtenha as listas HTTPS, SOCKS5 e HTTP das fontes acima.
2. Remova os IPs duplicados.
3. Teste a conectividade fazendo requisições leves (GET/HEAD) contra:
   - https://www.gstatic.com/generate_204 (Retorno rápido 204)
   - https://example.com (Página estática leve)
   - https://www.google.com (Latência global)
4. Classifique os tempos de resposta obtidos:
   - RÁPIDO: Menos de 1500 ms (Ideal para navegação normal)
   - MÉDIO: Entre 1500 ms e 4000 ms
   - LENTO: Entre 4000 ms e 8000 ms (Pode sofrer time-outs frequentes)
   - INVÁLIDO: Acima de 8000 ms ou sem resposta (Descartar imediatamente)
5. Salve localmente apenas os proxies validados com suporte total a conexões HTTPS.
